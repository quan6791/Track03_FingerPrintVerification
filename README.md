# ECCV18_challenge
This is the submission code for ECCV challenges:

the challange 03: Denoising and inpainting for fingerprint verification

The structure of challenge files:

       --challenge
       
              -- models 
       
              -- source
               
                      --files
               
                      --predict.bash
       
              -- data_set
       
       
To run the code:
```sh
$ chmod +x run.bash
$ chmod +x predict.bash
$ ./run.bash
```
or
```sh
$./predict.bash
```
